# 📦 Supply Chain Analytics Project  

## 🚀 Overview  
This project focuses on analyzing supply chain data to identify inefficiencies, optimize operations, and generate actionable business insights.  

It simulates a real-world scenario where data is processed and analyzed using **SQL, Python, and Power BI**.  

---

## 🔧 Tech Stack  
- Python – Data Processing  
- SQL – Data Analysis & Querying  
- Power BI – Dashboard & Visualization  
- Excel – Data Cleaning  

---

## ⚙️ Workflow  

### 1. Data Cleaning (Excel)  
- Handled missing values and inconsistencies  
- Structured dataset for analysis  

### 2. Data Processing (Python)  
- Prepared dataset for further analysis  
- Converted and structured data  

### 3. Data Analysis (SQL)  
- Performed advanced queries to extract insights  
- Identified risks, inefficiencies, and trends  

### 4. Data Visualization (Power BI)  
- Built an interactive dashboard  
- Visualized key KPIs and performance metrics  

---

## 📊 Key Analysis Performed  

- **Stockout Risk Analysis**  
  Identified warehouses with high stockout probability and negative inventory  

- **Delivery Delay Analysis**  
  Detected warehouses with above-average delivery delays  

- **Cost vs Sales Analysis**  
  Found warehouses with high logistics cost but low sales  

- **Capacity Utilization**  
  Measured warehouse utilization efficiency  

- **Demand vs Inventory Gap**  
  Highlighted imbalance between demand and stock  

- **Profitability Analysis**  
  Identified loss-making warehouse-product combinations  

- **Supplier Performance Analysis**  
  Evaluated suppliers based on delays and lead time  

---

## 📊 Key Insights  

- Some warehouses show **high stockout risk + negative inventory**  
- Certain locations have **higher delivery delays than average**  
- Multiple warehouses operate at **high cost but low revenue**  
- Demand-supply imbalance exists across product categories  
- Supplier delays significantly impact supply chain efficiency  

---

## 💡 Business Impact  

This analysis helps in:  
- Improving **inventory management**  
- Reducing **logistics costs**  
- Optimizing **warehouse utilization**  
- Enhancing **supplier performance**  
- Supporting **data-driven supply chain decisions**
