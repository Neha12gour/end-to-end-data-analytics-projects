# 📊 Legal Firm Data Analysis Project  

## 🚀 Overview  
This project demonstrates an end-to-end data analytics workflow where raw legal firm data is transformed into meaningful business insights.  

The project focuses on combining **data cleaning, transformation, cloud integration, and visualization** to simulate a real-world analytics pipeline.  

---

## 🔧 Tech Stack  
- Excel – Data Cleaning & Analysis  
- Python – Data Transformation (JSON Conversion)  
- Firebase – Realtime Database (Cloud Storage)  
- SQL – Data Querying  
- Power BI – Dashboard & Data Visualization  

---

## ⚙️ Workflow  

### 1. Data Cleaning (Excel)  
- Removed duplicates and handled missing values  
- Standardized city names and categories  
- Structured dataset for further processing  

### 2. Data Transformation (Python)  
- Converted cleaned dataset into JSON format  
- Prepared data for cloud upload  

### 3. Cloud Integration (Firebase)  
- Uploaded JSON data into Firebase Realtime Database  
- Created a scalable backend for analysis  

### 4. Data Connection (Power BI)  
- Connected Firebase database using API  
- Built an interactive dashboard  

### 5. Data Analysis (Excel + SQL)  
- City-wise analysis using Pivot Tables  
- Category distribution analysis  
- Regression-based trend analysis  
- KPI calculations  

---

## 📊 Key Insights  

- **Top Cities:** Bangalore, Vadodara, and Delhi dominate legal firm presence  
- **Category Insight:** Civil law firms form the majority  
- **Digital Gap:** Very few firms have websites → major opportunity for digital transformation  
- **Market Variation:** Different cities show different trends → localized strategies required  

---

## 💡 Business Impact  

This project highlights:  
- Opportunities for legal firms to improve their **digital presence**  
- Potential for **market expansion in less competitive cities**  
- Importance of **data-driven decision making**
